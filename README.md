## Description

A simple client-server web app that, updates simulatated gold prices live at 5sec intervals.<br>
Gold prices are simulated, since most API's take a long time to update prices, this way we can see the server-client in action.

Frontend: Vanilla JS <br>
Backend : Node.js

## Installation

1. Clone the repository: 
```bash
git clone <repo-url>
cd live-gold-price-app
```
2. Build the project:
```bash
npm start
```
3. Open `http://localhost:8080/` on preferred browser
